# Conversor MP4 para MP3

Projeto desenvolvido para atender uma necessidade da igreja: converter vídeos em formato MP4 para áudio MP3 de maneira simples, rápida e prática. A ideia é facilitar o aproveitamento das ministrações, permitindo extrair apenas o áudio dos cultos e disponibilizá-lo com mais facilidade para os membros.

## Como usar

### Executável local

Use o conversor diretamente no seu computador:

- **Com Python:** dê duplo clique em `converter.bat` para abrir a interface gráfica.
- **Executável .exe:** execute `build.bat` para gerar o `Conversor-MP4-MP3.exe` na pasta `dist\`. Depois, use o `.exe` em qualquer lugar.

> **Requisito:** FFmpeg deve estar instalado e no PATH do sistema para a conversão funcionar.

### Versão em rede

Para usar pela internet, acesse:

**https://convers-o-mp4-para-mp3-production.up.railway.app**

Basta enviar seu arquivo MP4 e fazer o download do MP3 convertido.
